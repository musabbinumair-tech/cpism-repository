HTML


What is html?   Why are we using it?

HTML (HyperText Markup Language) is the standard language for creating web pages. It structures content for display in browsers.


What is Tag?

A tag is the markup used to create elements in HTML, typically consisting of an opening and a closing part.


Difference between Tags / Elements:

Tags: The markup used to create elements, typically consisting of an opening and closing part

Elements: The complete structure that includes the opening tag, content, and closing tag


Difference between Attributes / Entities:

Attributes: Additional information within an HTML tag that modifies an element

Entities: Special codes that represent characters in HTML


Types of Attributes:

Types of attributes in HTML include:

Global Attributes: Can be used on any element

Specific Attributes: Unique to certain elements

Boolean Attributes: Present or absent, no value needed
